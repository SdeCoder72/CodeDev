Styling In CSS

Styling in CSS allows web developers to control the look and feel of HTML elements, including layout, colors, typography, and responsiveness—making web pages visually appealing and user-friendly.

Table Of Contents

  • 1. Introduction
  • 2. Ways to Add CSS
  • 3. Common Properties
  • 4. CSS Selectors
  • 5. Benefits of CSS
  • 6. Sample Code

1. Introduction

CSS stands for Cascading Style Sheets. It separates content (HTML) from presentation, allowing centralized design control across multiple pages.

2. Ways to Add CSS

  • Inline: Styles written in the HTML element's style attribute.
  • Internal: Styles placed within a <style> tag in the HTML <head>.
  • External: Styles in a separate .css file linked to HTML using <link>.

3. Common Properties

  • color: Text color
  • background-color: Background color
  • font-family: Font type
  • font-size: Text size
  • margin: Space outside element
  • padding: Space inside element
  • border: Borders

4. CSS Selectors

  • Element Selector: Targets by tag (e.g., p, h1)
  • Class Selector: Targets by class (e.g., .container)
  • ID Selector: Targets by ID (e.g., #heading)

5. Benefits of CSS

  • Centralized control of design
  • Consistent site-wide styling
  • Easier updates and maintenance

6. Sample Code


<!-- HTML -->
<h1 class="main-heading">Hello CSS!</h1>

<!-- CSS (Internal or External) -->
.main-heading {
    color: #4CAF50;
    font-family: Arial, sans-serif;
    border-bottom: 2px solid #333;
    padding-bottom: 10px;
}
