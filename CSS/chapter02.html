CSS Box Model

The CSS Box Model describes how HTML elements are rendered as rectangular boxes consisting of content, padding, border, and margin. Understanding the box model is crucial for precise layout and spacing in web development.

Table Of Contents

  • 1. Introduction
  • 2. Components
  • 3. Box Model Calculations
  • 4. HTML Example
  • 5. Content-box vs Border-box

1. Introduction

Every element in HTML is visualized as a box, which includes the content itself, and surrounding layers for padding, border, and margin.

2. Components

  • Content: The area that displays text, images, or other media.
  • Padding: Transparent space between content and border.
  • Border: Surrounds content and padding, giving visual edge.
  • Margin: Space outside the border, separates elements.

3. Box Model Calculations

For box-sizing: content-box: Total width = content width + padding + border. For box-sizing: border-box, total width is exactly the set width, including padding and border.

Property Description
Content Actual text or images inside the box
Padding Space between content and border
Border Line surrounding the padding and content
Margin Space outside the border

4. HTML Example

<div class="box-model-example">
  This div demonstrates the CSS Box Model.
</div>

<style>
.box-model-example {
  width: 300px;
  padding: 20px;
  border: 5px solid #2196f3;
  margin: 25px;
  background-color: #e3f2fd;
  box-sizing: border-box; /* Try 'content-box' for difference! */
}
</style>

5. Content-box vs Border-box

box-sizing Behavior Usage
content-box (default) Width/Height apply only to content area. Classic model; dimensions increase with border/padding.
border-box Width/Height include content, padding, border. Consistent sizing; preferred for layouts.
