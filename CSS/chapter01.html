CSS Fundamentals

CSS (Cascading Style Sheets) defines the look and feel of web pages by controlling layout, colors, fonts, and other visual elements. It separates content from presentation, making websites attractive and responsive.

1. What is CSS?

CSS stands for Cascading Style Sheets and is used to style web pages by applying designs to HTML elements. It controls elements' colors, spacing, layout, and responsive behavior.

2. Syntax & Structure

A CSS rule consists of a selector and a declaration block:

selector { property: value; }

  • Selector: Targets the HTML element (e.g., h1, .class, #id).
  • Property: The style attribute to change (e.g., color).
  • Value: The setting for the property (e.g., blue).

3. Selectors

  • Element Selector: Targets by HTML tag name (e.g., p).
  • Class Selector: Targets by class (e.g., .box).
  • ID Selector: Targets by id (e.g., #main).
  • Group & Descendant: div p targets p inside div.

4. Box Model

Every element is modeled as a box with content, padding, border, and margin:

  • Content: The actual text or image.
  • Padding: Space around the content.
  • Border: Wraps the padding (and content).
  • Margin: Space outside the border.

5. Applying CSS

  • Inline: Use the style attribute directly on the HTML element.
  • Internal: Place CSS inside a <style> tag in the HTML <head>.
  • External: Link a separate .css file with <link> in the <head>.

6. CSS Properties Table

Property Example Value Description

background-color #f3f3f3 Background fill color
margin 16px Outside space of element

border 1px solid #333 Element border style
font-size 18px Text size
display flex / block Layout type of element
