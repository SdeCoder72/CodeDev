CSS Animations

CSS animations allow web developers to create smooth transitions and lively effects by gradually changing styles of elements using keyframes, all without JavaScript. Animations enhance interactivity and draw attention to important components on a page.

Table Of Contents

  • 1. Introduction
  • 2. CSS Animation Principles
  • 3. Animation Properties
  • 4. Practical Examples
  • 5. CSS Transition vs Animation

1. Introduction

CSS animations use @keyframes to define style changes and special animation properties to control timing, iteration, and effect. They are widely used for attention-grabbing effects and subtle UX improvements.

2. CSS Animation Principles

  • Define @keyframes for animation steps
  • Bind animation to element via CSS property
  • Control timing, duration, direction, and repeats
  • Animate multiple properties (color, transform, opacity)

3. Animation Properties

  • animation-name: Name of the keyframes rule
  • animation-duration: How long one animation cycle runs
  • animation-timing-function: Speed curve (ease, linear)
  • animation-delay: Wait time before animation starts
  • animation-iteration-count: How many times to loop
  • animation-direction: Play forward, reverse, alternate
  • animation-fill-mode: How element looks before/after animation

4. Practical Examples

Below is a simple animation that stretches a button's width from 100px to 200px:

<div class="animated-btn">Click Me</div>

<style>
@keyframes stretch {
  0%   { width: 100px; }
  100% { width: 200px; }
}
.animated-btn {
  background: #4b81fa;
  color: #fff;
  padding: 1rem;
  border-radius: 5px;
  width: 100px;
  text-align: center;
  animation: stretch 1s ease-in-out 0s 1 forwards;
}
</style>

Other common uses: fade-in elements, slide navigation, bounce images, loading spinners.

5. CSS Transition vs Animation

Feature CSS Transition CSS Animation
Trigger User action (hover, click) Automatic, timeline based
Steps From one state to another Multiple steps via keyframes
Looping No looping Looping via iteration count
Complexity Simple, limited Complex, versatile
Properties transition-property, duration animation-name, duration, etc.
Usage Button hover, link color fade Loader, banners, complex UI effects
