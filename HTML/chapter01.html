HTML Basics & Structure

HTML (HyperText Markup Language) provides the foundation of all web pages. It defines the structure, layout, and elements on any website.

Table Of Contents

  • 1. What is HTML?
  • 2. HTML Document Structure
  • 3. Common HTML Tags

1. What is HTML?

HTML is a markup language that structures web content with tags, making text, images, links, and other elements readable by browsers.

2. HTML Document Structure

  • <!DOCTYPE html>: Declares the HTML version.
  • <html>: Wraps the whole document.
  • <head>: Holds meta info, styles, page title, links.
  • <body>: Contains all visible page content.

3. Common HTML Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraphs
  • <a>: Links
  • <img>: Images
  • <ul>, <ol>, <li>: Lists

4. Sample HTML Code


<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>Welcome to HTML learning.</p>
  </body>
</html>

5. Structure Table

Tag Description Visible?
<!DOCTYPE html> Declares HTML version No
<html> Root element No
<head> Meta info, links, title No
<title> Page title (tab name) Tab
<body> Main content Yes
