Tables and Forms in HTML

Tables display data in structured rows and columns, while forms collect user input for further processing. Both are essential for organizing content and interaction on a webpage.

1. Tables Introduction

A table in HTML organizes data into rows and columns, making viewing and comparison easy. They use the <table>, <tr>, <th>, and <td> tags to define structure.

2. Table Structure

  • <table>: The main container for table rows and columns.
  • <tr>: Defines a row in the table.
  • <th>: Marks a header cell (bold & centered by default).
  • <td>: Represents a standard cell for data.

4. Forms Introduction

Forms allow users to provide input to a website. HTML forms use various tags like <form>, <input>, and <label> to collect and process data.

5. Form Elements

  • <form>: Wraps the entire form.
  • <label>: Provides a text label for inputs.
  • <input>: Various types like text, email, password, etc.
  • <textarea>: Large text input area.
  • <select>: Dropdown selector.
  • <button>: Used for submission or action.
