Semantic HTML & Multimedia in HTML

Semantic HTML uses meaningful tags to represent content structure, while multimedia tags allow embedding audio, video, and images directly in a webpage.

Table Of Contents

  • 1. Semantic HTML
  • 2. Multimedia in HTML
  • 3. Key Semantic Elements
  • 4. Key Multimedia Elements

1. Semantic HTML

Semantic HTML clearly defines the meaning of elements on a webpage, improving accessibility, SEO, and maintainability.

For example, tags like <header>, <nav>, <main>, <article>, and <footer> express the structure and content purpose.

2. Multimedia in HTML

Multimedia in HTML refers to embedding images, audio, and video files directly inside HTML using special tags without the need for plugins.

This enhances web experiences with native controls for playback and customization.

3. Key Semantic Elements

  • <header>: Introductory content or navigation.
  • <nav>: Navigation links.
  • <main>: Main content of the document (only one per page).
  • <section>: Thematic grouping of content.
  • <article>: Independent, self-contained content.
  • <footer>: Footer of a document or section.

4. Key Multimedia Elements

The <audio> element embeds audio with built-in controls.

The <video> element embeds video content with playback controls.
