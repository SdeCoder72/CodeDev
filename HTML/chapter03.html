Links and Images in HTML

Learn how to use links and images to create interactive and visually engaging web pages. HTML uses special tags to insert hyperlinks and images, making websites more useful for navigation and presentation.

Table Of Contents

  • 1. HTML Links
  • 2. HTML Images
  • 3. Image as Link
  • 4. Feature Comparison

1. HTML Links

Hyperlinks are created with the <a> tag. Use the href attribute to specify the link's destination.

<a href="https://www.google.com">Visit Google</a>

2. HTML Images

To display images, use the <img> tag. The src attribute sets the file path or URL, while alt describes the image for accessibility.

<img src="cat.jpg" alt="A cute cat" width="200" height="150">

3. Image as Link

Combine links and images by nesting an <img> inside an <a> tag. This makes the image clickable.

<a href="https://www.example.com">
  <img src="logo.png" alt="Example Logo">
</a>
