HTML Text Formatting

HTML text formatting refers to the use of specific HTML tags to modify the appearance and structure of text on a webpage. It allows you to style text in different ways, such as making it bold, italic, underlined, highlighted, or struck-through.

  • 3. Logical Tags

1. Introduction

HTML formatting tags are used to style textual content and provide semantic meaning, affecting both the appearance and accessibility of web pages.

2. Formatting Categories

  • Logical Tags
  • Physical Tags

3. Logical Tags

Logical tags indicate semantic meaning or importance, without necessarily affecting the style:

  • <strong> – Emphasizes strong importance (Strong Text)
  • <em> – Stresses emphasis (Emphasized text)
  • <mark> – Highlights background (Marked text)

4. Physical Tags

Physical tags directly change the look of text for styling purposes:

  • <b> – Bold (Bold text)
  • <i> – Italic (Italic text)
  • <u> – Underline (Underlined text)
  • <small> – Small size (Small text)
  • <sub> – Subscript (H2O)
  • <sup> – Superscript (E=mc2)
  • <del> – Deleted (Deleted text)
  • <ins> – Inserted (Inserted text)

6. Logical vs. Physical Tags

Tag Logical (Semantic) Physical (Styling)
<strong> Yes Bold
<b> No Bold
<em> Yes Italic
<i> No Italic
<mark> Yes Highlighted
<u> No Underlined
<del> Yes Strikethrough
