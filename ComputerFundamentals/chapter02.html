Computer Hardware

Computer hardware refers to the physical components of a computer that you can see and touch. These components work together to process input and deliver output based on user instructions.

1. Introduction

Computer hardware encompasses all **physical devices** of a computer system. These work together to process data and execute instructions from users.

2. Hardware Categories

  • Input Devices
  • Output Devices
  • Storage Devices
  • Internal Components

3. Input Devices

Devices allowing users to send commands or data to the computer. Popular examples include:

  • Keyboard (Alphabet, number, function keys, often wireless)
  • Mouse (Pointer/cursor control, click, drag)
  • Scanner (Digitizes documents/images)
  • Trackball (Stationary pointer control)
  • Light Pen (Draw/select on CRT screens)
  • Microphone (Audio input/voice commands)
  • OCR & Bar Code Reader (Text/Code data input)

4. Output Devices

Devices that display or produce output for the user:

  • Monitor (Visual display)
  • Printer (Text & images on paper)
  • Plotter (Large blueprints and graphics)
  • Speakers (Sound output)

5. Storage Devices

Devices for storing data permanently or temporarily:

  • CDs (Rom, Recordable, Rewritable; up to 700MB)
  • DVDs (ROM, R/RW; from 4.7GB to 17GB)
  • Hard Disk (Internal/External, GB to PB capacity)

Also see: RAM (temporary memory), ROM (read-only memory)

6. Hardware Components

  • CPU (Central Processing Unit): Control, ALU, Memory units
  • Motherboard: Main circuit board connecting everything
  • RAM: Temporary, volatile memory (SRAM, DRAM)
  • VGA Port: Video display connection
  • Power Supply: Powers all components
  • Cooling Fan: Prevents system overheating
  • Hard Drive: Permanent storage for files/programs

7. Hardware vs. Software

Category Hardware Software
Definition Physical devices Programs/applications
Examples CPU, RAM, Monitor, Keyboard Operating System, Browser, Video Editor
Purpose Executes & stores physically Processes data, user interface
Dependence Independent, but needs software Needs hardware to work
Interaction Direct (user/system) Indirect (uses hardware)
Upgradability Add/replace physical parts Update/patch/new version
Cost Higher initial cost Lower initial (generally)
Installation Physical setup required Installed via software packages
